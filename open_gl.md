## Open GL
The source code in this repository is the result of study from [http://www.opengl-tutorial.org](http:://www.opengl-tutorial.org) and cover the following topics:

* 01 - Create window
* 02 - Draw a Triangle
* 03 - Matrix projection
* 04 - Color Cube
* 05 - Texture Cube
* 06 - Keyboard Mouse
* 07 - Model Loading
* 08 - Basic Shading
* 09 - Vertex Buffer Object
* 10 - Transparency
* 11 - Text
* 13 - Normal Mapping
* 14 - Render to texture
* 15 - Lightmaps
* 16 - Shadow mappings
* 17 - Rotations
* 18 - Particles

Git Hub repository link [OpenGL](https://github.com/NelsonBilber/cg.opengltutorial)
